# App icon

Place your app icon here:

- **Filename:** `app_icon.png`
- **Size:** 1024Ã—1024 px (recommended)
- **Format:** PNG, no transparency for best results on Android adaptive icon

Then run:

```bash
dart run flutter_launcher_icons
```

This generates all required sizes for Android and iOS. Replace this file with your own icon to change the launcher icon.
